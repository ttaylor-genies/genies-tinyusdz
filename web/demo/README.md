## Setup

### Build TinyUSDZ wasm module

See ../README.md for details.

## Run locally

Copy tinyusdz.js and tinyusdz.wasm from ../dist

Edit tinyusdz.js path to local file path in `main.js`

Run the server(We use bun + vite).

```
$ bun run dev
```
